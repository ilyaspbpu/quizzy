# QuizRoom MVP

Веб-приложение для проведения квизов в реальном времени.

## Возможности

- регистрация и вход для участников и организаторов;
- личный кабинет для обеих ролей;
- конструктор квизов с категориями, таймером и правилами;
- вопросы с текстом и изображением через drag-and-drop;
- одиночный и множественный выбор ответа;
- запуск комнаты по коду;
- синхронизация текущего вопроса, таймера, ответов и таблицы участников через Socket.IO;
- подсчет баллов с бонусом за скорость;
- история проведенных и пройденных квизов.

## Тестовые аккаунты

- организатор: `admin@example.com`, пароль `password`;
- участник: `user@example.com`, пароль `password`.

## Стек

- React;
- Vite;
- Node.js;
- Express;
- Socket.IO;
- `crypto.scrypt` для хеширования паролей.

## Модель данных

- `User`: имя, email, роль `organizer` или `participant`, хеш пароля.
- `Quiz`: организатор, название, категории, правила, лимит времени, список вопросов.
- `Question`: текст, изображение, режим выбора `single` или `multiple`, варианты ответа.
- `Session`: код комнаты, статус, текущий вопрос, участники, ответы, баллы, дата завершения.
- `Answer`: выбранные варианты, корректность, начисленные баллы, время ответа.

## Запуск

```bash
npm install
npm run dev
```

Клиент: `http://127.0.0.1:5173`

API и WebSocket: `http://127.0.0.1:4000`


## Основные этапы разработки

1. Спроектированы регистрация, кабинет,  запуск комнаты, подключение участника, прохождение квиза, таблица участников.
2. Реализован REST API для аккаунтов, квизов и истории.
3. Реализован WebSocket-слой: комнаты, состояние в реальном времени, таймер, прием ответов только во время показа вопроса.
4. Реализован интерфейс организатора: настройки квиза, вопросы с текстом или изображением, одиночный и множественный выбор.
5. Реализован интерфейс участника: вход по коду комнаты, ответы, отображение статуса и баллов.
